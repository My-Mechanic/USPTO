Notifications & reminders

Get desktop alerts when a tracked patent/trademark changes status or a filing deadline approaches. Alerts fire while this site is open and auto-checks. Background email alerts are handled by the GitHub Actions (see README).

Want background email alerts for patents too (even when this site is closed)? Export patent-watchlist.json, commit it to data/patent-watchlist.json, and add your ODP key as the ODP_API_KEY repo secret. The daily Monitor patents Action then emails you on any change. Off until you opt in.

USPTO’s site blocks direct browser access (no CORS), so instant lookups go through a proxy. For fast, reliable results, deploy the free Cloudflare Worker in workers/tsdr-proxy.js and paste its URL below. Leave blank to use best-effort public proxies.
